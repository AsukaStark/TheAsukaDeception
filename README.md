# TheAsukaDeception

## 项目简介

本项目为纪实作品，详尽记录并分析了网络人物 **Asuka**（别名：硅谷柜姐）从长期网暴、实施杀猪盘、种族歧视、群体网暴到反控性侵、自我包装成受害者的一系列操作。

本书基于大量证据——包括受害者证词、公开发言截图、法院文件与线上社交行为轨迹——剖析了一个反社会型人格如何在当今社交媒体时代构筑起虚假的道德高地，并如何利用性别与身份光环操控舆论，伤害他人。
